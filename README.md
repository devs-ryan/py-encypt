# py-encypt
python text file encypt / decypt progam

Usage:
  ./cryptor.py [option] [flag]
Options:
  ------------
  [1] Encypt:
  ------------
  Option = encrypt
  Description: Encrypts a file
  Flags: N/A
  ------------
  [2] Decrypt:
  ------------
  Option = decrypt
  Description: Decrypts a file
  Flags:
    --output
      (Will request an output file rather than printing to terminal)
